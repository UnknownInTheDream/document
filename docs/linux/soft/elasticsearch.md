> elasticsearch 版本 7.6.2

1. 下载elasticsearch安装包

在 /usr/local 目录下进行

```
wget https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-7.6.2-linux-x86_64.tar.gz
```

2. 解压elasticsearch

```
tar -zxvf elasticsearch-7.6.2-linux-x86_64.tar.gz
```
